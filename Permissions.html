Permissions:

securelogin.* : Grants access to all of SecureLogin.

  • Default: Op

securelogin.nofreeze : Allow a player not to have to log in before playing.

  • Default: Op

securelogin.reload : Allow access to /SlReload.

  • Default: Op

securelogin.setloginspawn : Allow access to /SetLoginSpawn.

  • Default: Op

securelogin.loginspawn : Allow access to /LoginSpawn

  • Default: Op

securelogin.loginspawn.other : Allow access to /LoginSpawn <player>

  • Default: Op

securelogin.help : Allow access to /SecureLogin (/sl) to see the help page.

  • Default: Everyone

securelogin.seespawnwarning : Allow someone to be alerted that the Login Spawn has not been set if it hasn't been set.

  • Default: Op

securelogin.login : Allow access to /Login.

  • Default: Everyone

securelogin.register : Allow access to /Register.

  • Default: Everyone

securelogin.createsecurityquestion : Allow access to /CreateSecurityQuestion.

  • Default: Everyone

securelogin.getsecurityquestion : Allow access to /GetSecurityQuestion.

  • Default: Op

securelogin.changesecurityquestion : Allow access to /ChangeSecurityQuestion.

  • Default: Everyone

securelogin.changesecurityquestion.other : Allow access to /ChangeSqFor.

  • Default: Op

securelogin.createsecurityanswer : Allow access to /CreateSecurityAnswer.

  • Default: Everyone

securelogin.getsecurityanswer : Allow access to /GetSecurityAnswer

  • Default: Op

securelogin.changesecurityanswer : Allow access to /ChangeSecurityAnswer.

  • Default: Everyone

securelogin.changesecurityanswer.other : Allow access to /ChangeSaFor

  • Default: Op

securelogin.getpassword : Allow access to /GetPassword.

  • Default: Op

securelogin.changepassword : Allow access to /ChangePassword.

  • Default: Everyone

securelogin.changepassword.other : Allow access to /ChangePasswordFor

  • Default: Op

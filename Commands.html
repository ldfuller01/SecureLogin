Commands:

/SecureLogin : See the help page for this plugin.

  • Alias: /Sl

/Login : Log into the game with your password.

/Register : Register your password to be able to log into the game.

/SetLoginSpawn : Set the location to where players are teleported to log in.

/LoginSpawn [Player] : Without Player input, teleports you to the place where players go to log in. With player input, teleports the specified player to the Login Spawn.

/SlReload : Reload the plugin's configuration.

/CreateSecurityQuestion : Create your Security Question that you may use to change your password, old Security Question, or your Security Answer, as defined by the config file.

  • Alias: /CreateSQ

/GetSecurityQuestion <Player> : View someone's Security Question.

  • Alias: /GetSQ

/ChangeSecurityQuestion : Change your Security Question.

  • Alias: /ChangeSQ

/ChangeSqFor <Player> : Change someone else's Security Question.

/CreateSecurityAnswer : Create the answer to your Security Question.

  • Alias: /CreateSA

/GetSecurityAnswer <Player> : View someone else's Security Answer.

  • Alias: /GetSA

/ChangeSecurityAnswer : Change your Security Answer.

  • Alias: /ChangeSA

/ChangeSaFor <Player> : Change someone else's Security Answer.

/GetPassword <Player> : View someone else's login password.

/ChangePassword : Change your login password.

/ChangePasswordFor <Player> : Change someone else's login password.
